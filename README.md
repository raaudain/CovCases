# CovCases
[CovCases](https://covcases.raaudain.vercel.app/) an interactive map application populated with the number of COVID-19 cases in the United States.

## APIs Used

* Google Maps API
* NovelCOVID API

## Packages Used

* React
* axios
* use-places-autocomplete
* @react-google-maps/api
* @reach/combobox
